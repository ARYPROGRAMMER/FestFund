## 🚀 **Quick Start**

```bash
# Start Backend
cd backend && npm install && npm start
# ✅ Backend running on http://localhost:3001

# Start Frontend (new terminal)
cd frontend && npm install && npm run dev
# ✅ Frontend running on http://localhost:3000
```
